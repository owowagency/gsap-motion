## Example code

Simple Example

```js
import { Marquee } from "@owowagency/gsap-motion";

const marquee = new Marquee("#marquee");
```

Advanced example

```js
/**
 * A marquee with speed 2, that bases its motion direction on scroll direction,
 * and responds to scroll gestures with a velocity factor of 0.01.
 */
const marquee = new Marquee("#marquee", {
  speed: 2,
  velocity: 0.01,
  direction: "scroll",
  onCreated() {
    console.log("Created!");
  },
  onUpdate(progress) {
    console.log("tick", progress);
  },
});
```
