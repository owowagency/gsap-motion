## Example code

```js
import gsap from "gsap";
import { Pointer } from "@owowagency/gsap-motion";

gsap.ticker.add(() => {
  // do something with pointer position...
  console.log({
    x: Pointer.instance.clientX,
    y: Pointer.instance.clientY,
  });
});
```

## Properties

### Pointer.instance.clientX

The pointer's horizontal coordinate within the application's viewport.

[See MouseEvent: clientX](https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent/clientX)

### Pointer.instance.clientY

The pointer's vertical coordinate within the application's viewport.

[See MouseEvent: clientY](https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent/clientY)

### Pointer.instance.normalX

The pointer's normalized horizontal coordinate, relative the application's viewport. Normalized coordinates range from 0 to 1, representing the percentage of the total width or height.

### Pointer.instance.normalY

The pointer's normalized vertical coordinate, relative the application's viewport. Normalized coordinates range from 0 to 1, representing the percentage of the total width or height.

### Pointer.instance.viewWidth

The application's viewport width.

[See Window: innerWidth](https://developer.mozilla.org/en-US/docs/Web/API/Window/innerWidth)

### Pointer.instance.viewHeight

[See Window: innerHeight](https://developer.mozilla.org/en-US/docs/Web/API/Window/innerHeight)
